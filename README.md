##Crowd monitoring system
## Features

- **Real-Time Crowd Detection:** Developed a smart surveillance system using **YOLOv8 Nano** and **OpenCV** to detect and count people from live CCTV/IP camera feeds with high accuracy and low latency.

- **Crowd Density Analysis:** Implemented adaptive grid-based crowd density estimation to identify high-density zones and classify crowd levels into **Low**, **Medium**, and **High** for effective monitoring.

- **Full-Stack Web Dashboard:** Built a responsive dashboard using **React.js**, **Node.js**, **Python**, and **MongoDB** to display live video streams, real-time people counts, crowd density, and analytics.

- **Interactive Campus Monitoring:** Integrated a live campus map that visualizes the crowd status of multiple surveillance locations, enabling security personnel to quickly identify congested areas.

- **Automated SMS Alerts:** Integrated **Twilio SMS API** to automatically notify security personnel whenever crowd density exceeds predefined safety thresholds for a sustained period.

- **Scalable Multi-Camera Surveillance:** Designed a scalable architecture supporting multiple cameras, REST APIs, historical analytics, image uploads for blind spots, and centralized monitoring for comprehensive area-wide crowd management.
